PMA API Calculator
     Aplikace je funkční kalkulačka pracujicí čistě za pomoci API. Jedná se o zdarma API bez nutnosti registrace. Omezený čas pro jeden výpočet je deset vteřin a denně je možno zavolat 10000 požadavků. Kalkulačka umožňuje základní aritmetické operace, ale i goniometrické funkce, umocňování, faktoriál a další.
